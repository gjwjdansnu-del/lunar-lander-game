# Lunar Lander 🌙

달 착륙선을 조종하는 2D 물리 시뮬레이션 게임입니다.

## 플레이

[게임 플레이하기](https://apl.github.io/lunar-lander-game/) *(배포 후 URL 업데이트)*

### 조작

| 키 | 동작 |
|---|---|
| ↑ / W | 메인 추진 (연료 소모) |
| ← / A | 자세 제어 (반시계, 연료 소모) |
| → / D | 자세 제어 (시계, 연료 소모) |

### 목표

- 랜덤 생성된 달 지형 위 녹색 **착륙 지점**에 안전하게 착륙하세요.
- 수직 속도 2.5 m/s 이하, 수평 속도 1.5 m/s 이하, 자세 12° 이내로 착지해야 합니다.

### 물리

- 달 중력: 1.62 m/s²
- 강체 동역학: 운동량, 각운동량, 회전 모멘트 반영
- 추진력은 엔진 위치에서 적용되어 토크 발생
- 자세 제어 스러스터는 별도 연료 소모

## 로컬 실행

```bash
# 간단한 HTTP 서버로 실행
python3 -m http.server 8080
# http://localhost:8080 접속
```
